MySQL-mysql 8.0.11安装教程

1.下载安装包，解压，解压后如下图所示

2.配置环境变量

3.在安装目录下，新建my.ini文件，初始化配置

4.初始化数据库

在MySQL安装目录的 bin 目录下执行命令：

mysqld –initialize –console

注意！执行输出结果里面有一段：[Note][MY-010454] [Server] A temporary password is generated for root@localhost: 1HV)1?s,lYrP， 其中root@localhost:后面的“1HV)1?s,lYrP”就是初始密码（不含首位空格）。在没有更改密码前，需要记住这个密码，后续登录需要用到。

5.注册服务

mysqld install mysql

sc delete mysql

启动服务

net start mysql

6.修改root密码

在MySQL中执行命令：

ALTER USER ‘root‘@’localhost’ IDENTIFIED WITH mysql_native_password BY ‘新密码’;
